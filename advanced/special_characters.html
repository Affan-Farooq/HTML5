Using Special Characters in HTML

This document demonstrates how to use special characters and HTML entities to display symbols and reserved characters in web content.

HTML Entities

HTML entities are used to display reserved or special characters in HTML. They start with an ampersand (&) and end with a semicolon (;).

Examples of reserved characters:

  • Ampersand: &amp; → &
  • Less than: &lt; → <
  • Greater than: &gt; → >
  • Double quote: &quot; → "
  • Single quote: &apos; → '

Commonly Used Symbols

Here are some commonly used symbols and their corresponding HTML entities:

  • Copyright: &copy; → ©
  • Registered Trademark: &reg; → ®
  • Euro: &euro; → €
  • Pound Sterling: &pound; → £
  • Yen: &yen; → ¥
  • Degree: &deg; → °
  • Plus-Minus: &plusmn; → ±
  • Section: &sect; → §

Numeric and Hexadecimal Character References

HTML also supports numeric and hexadecimal references to represent characters. Here are a few examples:

  • Heart symbol (numeric): &#9829; → ♥
  • Star symbol (numeric): &#9733; → ★
  • Heart symbol (hexadecimal): &#x2665; → ♥
  • Star symbol (hexadecimal): &#x2605; → ★
